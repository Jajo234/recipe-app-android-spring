# Requisitos Funcionales

Este documento describe los requisitos funcionales del sistema Recipe App.

## Gestión de Usuarios
RF-01: El sistema permitirá el registro de usuarios mediante correo electrónico y contraseña.  
RF-02: El sistema permitirá al usuario iniciar sesión.  
RF-03: El sistema permitirá al usuario cerrar sesión.  
RF-04: El sistema permitirá la recuperación de contraseña.  

## Gestión de Recetas
RF-05: El sistema permitirá al usuario crear una receta.  
RF-06: El sistema permitirá visualizar una lista de recetas disponibles.  
RF-07: El sistema permitirá visualizar el detalle de una receta.  
RF-08: El sistema permitirá al usuario editar sus propias recetas.  
RF-09: El sistema permitirá al usuario eliminar sus propias recetas.  

## Interacción del Usuario
RF-10: El sistema permitirá al usuario guardar recetas como favoritas.  
RF-11: El sistema permitirá al usuario buscar recetas por nombre o ingrediente.  
RF-12: El sistema permitirá al usuario filtrar recetas por categoría.  
RF-13: El sistema permitirá al usuario calificar recetas.
